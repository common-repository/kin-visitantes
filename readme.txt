Track visitors to your website easily and effectively.

== Description ==

NOW WITH GOOGLE CHARTS!

Keep track of how many visitors reach your site, when they accessed it, what language they speak, what browser they use, where they came from, and what they looked at.

* Beautiful Google Charts presentation
* Dashboard widget
* Date and time visited
* Visitor language
* Visitor browser and operating system
* IP address
* Visited page
* Refering site

Learn more at http://www.kinwebdesign.com

== Installation ==

1. Upload the folder, 'kin-visitantes' and all its files to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
